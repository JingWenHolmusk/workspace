Module contents¶

Runs SQL commands to work with database tables to generate a master table which has all needed information.

The table will then be fetched and saved as data/intermediate/db.pickle . This is a very heavy module and is going to be replaced after the SQL-python-pgIO-psycopg2 wrapper thing is done.

"path"       : "modules/JWComorbid/preProcessDB/preProcessDB.py",
"execute"    : true,
"description": "",
"owner"      : ""
